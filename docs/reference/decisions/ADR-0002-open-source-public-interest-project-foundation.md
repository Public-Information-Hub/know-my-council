# ADR-0002: Open-Source Public-Interest Project Foundation

Status: accepted
Date: 2026-04-04

## Context

KnowMyCouncil is a public-interest civic transparency project. Trustworthiness, evidence discipline, and contributor-friendliness matter from day one.

## Decision

Establish a serious open-source foundation in the repository, including:

- a permissive open-source licence (MIT)
- code of conduct, contribution guidance, and security reporting process
- issue and PR templates suitable for civic transparency and evidence-sensitive work
- non-partisan, evidence-led positioning in documentation

## Consequences

- Contributors have predictable expectations for how to participate.
- Maintainers have a clearer baseline for review, moderation, and triage.
- The project signals credibility without claiming maturity it does not yet have.
