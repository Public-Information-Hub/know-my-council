# Support

KnowMyCouncil is an early-stage open-source project. Support is provided on a best-effort basis by maintainers and community volunteers.

## Where to ask questions

- GitHub Discussions (recommended, if enabled) for usage questions, design questions, and general help.
- GitHub Issues for bugs, concrete work items, and tracked proposals.

## Where to report what

- **Bug reports:** use the Bug report issue template.
- **Feature requests:** use the Feature request template.
- **Data issues:** use the Data issue template (missing data, broken sources, suspected incorrect records).
- **Investigation leads:** use the Investigation / lead template (patterns to research, anomalies to validate).

## What support is available

- Clarifying how the repository is structured
- Guidance on contribution workflow
- Triage help for issues that include enough context and sources

## What support is not available (yet)

- Guaranteed response times
- Dedicated troubleshooting for custom deployments
- Legal advice, journalistic advice, or FOI/EIR advice (see [DISCLAIMER.md](DISCLAIMER.md))
