## Responsibilities

Maintainers are expected to:

- review pull requests in a timely, best-effort way
- uphold evidence and provenance standards
- keep dependencies and security posture reasonable
- be respectful and enforce the code of conduct when needed
- document major decisions

## Access and permissions

Maintainership implies access that should be handled carefully:

- repo write access is given only when necessary
- sensitive reports should be handled via private channels (see [SECURITY.md](SECURITY.md))
